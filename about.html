Hello, I'm James

I'm a student at the University of South Florida, majoring in Information Science with a focus on Data Science and Analytics. My technical expertise includes SQL, Python, and R, which I use to uncover meaningful insights from data. Currently, I'm gaining valuable industry experience as a System Administration Intern, where I work with tools like Proxmox, Ubuntu, Microsoft Intune, and IT inventory management. As part of my role, I've developed internal tools using Django to increase efficiency across systems.

My passion for technology extends to self-hosting and infrastructure management, which led me to build my own homelab. Through this setup, I run services like Jellyfin for media streaming and various server applications using Proxmox, Docker, and Ubuntu Server.
